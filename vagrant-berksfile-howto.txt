cleanup
=======

gem uninstall vagrant

gem uninstall berkshelf

# if moving code between machines, remove the following file as it contains a filepath:
rm /.vagrant/machines/default/berkshelf


install
=======

install virtualbox

install ruby 1.9 (via rvm)

install vagrant binary

vagrant plugin install vagrant-berkshelf

vagrant plugin install vagrant-omnibus

bundle install



create
======

berks cookbook paulapp

cd paulapp

bundle install



config
======

edit Vagrantfile



run
===

vagrant up

vagrant provision
